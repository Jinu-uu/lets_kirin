https://nodejs.org/en/download/package-manager


Git 설치 링크

### 실행 방법

저장소 클론 터미널을 열고 다음 명령어를 입력하여 저장소를 클론합니다:

````
git clone https://github.com/사용자명/저장소명.git .
````


저장소명 폴더로 이동합니다:
```
cd 저장소명
```
의존성 설치 프로젝트의 의존성을 설치합니다:

````
npm i
````


프로젝트 시작 다음 명령어를 입력하여 프로젝트를 시작합니다:

````
npm start
````
